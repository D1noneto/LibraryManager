# LibraryManager

## Descrição
Sistema de controle de empréstimos de livros para bibliotecas pequenas ou pessoais, utilizando conceitos de Programação Orientada a Objetos em Python.

## Funcionalidades
- Cadastro de livros
- Empréstimo de livros
- Devolução de livros
- Controle de disponibilidade

python main.py
